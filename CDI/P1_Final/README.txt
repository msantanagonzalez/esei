Nota: Para ejecutar el programa se debe utilizar java –jar main.jar indicando una serie de parámetros, 
el primero para el número de ejercicio, y el segundo para el número de hilos. 

Para los apartados 2.X hace falta indicar un tercer parámetro entero, 
para el apartado 2.6 el tercer parámetro se recomienda que sea un numero entre 1 y 3(ambos inclusive) para observar mejor los gráficos.
